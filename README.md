# Adams Auto Service Website

A professional business site for Adams Auto Service, Grandview, MO.

## Features
- SEO-optimized, mobile-friendly
- Sections: Services, Testimonials, Booking, Contact
- Local business schema markup for Google

## Deployment
1. Host the files on Netlify, Vercel, GitHub Pages, or your choice of static hosting.
2. Edit `index.html` to customize services, testimonials, or contact info as needed.

## Booking Form
Form is demo/static. For real requests, integrate with a backend service or form endpoint (e.g. Formspree).
